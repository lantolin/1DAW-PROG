Hello world! Proyecto para ser usado como base para un proyecto Java

Características:

  • Maven, configurado para usar JUnit 5 y Java 17
  • El proyecto se llama ProyectoBase-id1, para usarlo tienes que poner tu id1 tanto en el nombre del proyecto como en el pom.xml
  • La clase que contiene el metodo main() que lanza la funcionalidad es App, eso esta configurado en pom.xml para ser la clase main del JAR
